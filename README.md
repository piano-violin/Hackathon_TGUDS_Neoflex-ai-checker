# Hackathon_TGUDS_Neoflex-ai-checker
Neoflex AI-Checker Автоматическая проверка тестовых заданий для позиции Data Engineer. - Оценка ответов по 10-балльной шкале через LLM Qwen2.5-Coder - Обратная связь для кандидатов и организаторов - Поддержка 9 форматов файлов - Telegram-бот и экспорт в Excel Python, PyTorch, PostgreSQL.
